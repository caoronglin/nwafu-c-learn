## 🚀 使用方法

### 方法1：使用启动脚本（推荐）

**Linux/Mac:**
```bash
cd website
./start.sh
```

**Windows:**
```batch
cd website
start.bat
```

### 方法2：手动启动

```bash
cd website
python3 -m http.server 8000
```

### 方法3：直接打开

用浏览器打开 `index.html` 文件

### 访问地址

- 主页：http://localhost:8000
- 案例：http://localhost:8000/pages/cases.html
