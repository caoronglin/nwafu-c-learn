# C语言程序设计教学网站

一个参考 csgraduates.com 风格创建的C语言程序设计教学网站，包含完整的课程体系、实战案例和考试面试准备内容。

## ✨ 特色功能

- 📚 **完整课程体系**：从基础到高级，10个章节全覆盖
- 💻 **丰富案例**：8个完整项目案例，含详细解析
- 📝 **考试准备**：易错知识点总结，高频考点清单
- 🎯 **面试必备**：常见面试题 + 算法详解
- 🎨 **简约设计**：界面清爽，专注内容
- 📱 **响应式**：支持桌面和移动设备

## 🚀 快速开始

### 方式1：直接运行（开发测试）

```bash
cd website
python3 -m http.server 8000
# 访问 http://localhost:8000
```

### 方式2：作为二级目录部署

```bash
cd website
chmod +x setup-secondary.sh
./setup-secondary.sh
# 按提示输入二级目录路径
```

### 方式3：使用启动脚本

**Linux/Mac:**
```bash
cd website
./start.sh
```

**Windows:**
```batch
cd website
start.bat
```

## 🔧 配置为二级目录

如果需要将本网站作为主站的二级目录使用：

### 快速配置

```bash
cd website
./setup-secondary.sh
# 输入二级目录路径，如：c-course
```

### 手动配置

1. 修改所有HTML文件中的CSS引用：
   ```html
   <link href="/your-path/css/main-simple.css">
   ```

2. 修改导航链接：
   ```html
   <a href="/your-path/pages/chapter2.html">
   ```

详细说明请参考：[SECONDARY_SITE_GUIDE.md](SECONDARY_SITE_GUIDE.md)
